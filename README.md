# real-account-book
- 나만의 가계부


### Commands
- run dev (frontend + server)
```
$ npm run dev
```

- run only frontend
```bash
# root directory
$ npm run dev:frontend
# or move '/frontend' directory
$ npm run dev
```

- run only server
```bash
# root directory
$ npm run dev:server
# or move '/server' directory
$ npm run dev
```
